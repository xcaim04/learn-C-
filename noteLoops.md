# Loops in C#

## WHILE SENTENCE
```
while (condition)
{
    // Code...
}
```

## DO WHILE SENTENCE
```
do
{
    // Code...
} while (condition);
```

## FOR SENTENCE
```
for (int i = 0; i < 10; i++)
{
    // Code...
}
```

## FOREACH SENTENCE
```
foreach (type variableName in arrayName) 
{
  // code block to be executed
}
```
